# Türkiye'de Dijital Yaşam

Bu proje, Türkiye'de çocuk ve gençlerin dijital yaşamını ve internet kullanım alışkanlıklarını görselleştiren bir web sunumudur.

## Yapılan güncellemeler

- `index.html` dosyası güncellendi.
- `Kullanım Sıklığı` grafiği yerine `Dijital Alışkanlıklar` görseli gösterilecek şekilde düzenlendi.
- `Cinsiyete Göre Analiz` bölümüne iki yeni grafik sekmesi eklendi:
  - `Dijital Alışkanlıklar` → `cinsiyet/dijital_alikanliklarimiz.png`
  - `Yaş & Cinsiyet` → `cinsiyet/dijital_alikanliklarimiz2.png`
- İnternet bölümüne yeni bir görsel eklendi:
  - `internet/dijital_alikanliklarimiz.png`

## Dosya yapısı

- `index.html` — proje ana sayfası ve interaktif grafik gösterimi.
- `internet/dijital_yasam_sankey.html` — internet akışını gösteren Sankey diyagramı.
- `internet/dijital_alikanliklarimiz.png` — internet bölümünde kullanılan yeni dijital alışkanlık görseli.
- `cinsiyet/dijital_alikanliklarimiz.png` — cinsiyet bölümüne eklenen yeni görsel.
- `cinsiyet/dijital_alikanliklarimiz2.png` — cinsiyet bölümüne eklenen ikinci yeni görsel.

## Kullanım

1. Bu klasördeki `index.html` dosyasını açın.
2. `İnternet Kullanımı` ve `Cinsiyete Göre Analiz` bölümlerindeki sekmelere tıklayarak görselleri inceleyin.
3. GitHub Pages üzerinde yayınlamak için değişiklikleri repository'ye ekleyip push edin.

## Not

Bu güncellemeler yerel olarak uygulandı. GitHub Pages sitesinde görünmesi için repository'yi commit ve push etmeniz yeterlidir.
